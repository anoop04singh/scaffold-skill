Installation demo

How to install in Claude Desktop

Follow this quick visual guide to upload the ZIP in Customize → Create Skill and activate Scaffold.

Path A: Claude Desktop (ZIP upload)

  1. Download the skill ZIP package.
  2. Open Claude Desktop and go to Customize → Create Skill.
  3. Drop the ZIP into the Upload Skill section.
  4. Invoke with /scaffold python (or any language) and upload your research PDF.

Path B: Manual install (folder copy)

  1. Download and unzip the skill package.
  2. Copy the folder to ~/.claude/skills/scaffold/ or project .claude/skills/scaffold/.
  3. Open Claude Code and invoke /scaffold python (or any language).
  4. Upload a research PDF and receive implementation-ready outputs.

Use these commands in Claude to run Scaffold:

/scaffold languageYouPrefer
for example:
/scaffold rust
/scaffold solidity
/scaffold typescript
or simply:
/scaffold
