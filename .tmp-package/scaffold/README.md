# scaffold skill

Turn a research paper into an implementation roadmap and starter code skeleton.

## Invoke

- `/scaffold python`
- `/scaffold rust performance`
- `/scaffold solidity security`

## Expected input

- Upload a paper PDF or paste relevant sections.
- Optionally pass target language and focus as arguments.

## Output

1. Core Contribution
2. Key Primitives
3. Dependency Map
4. Protocol Steps
5. Pseudocode
6. Starter Scaffold
7. Implementation Roadmap
8. Risks and Validation
9. Clarifying Assumptions
